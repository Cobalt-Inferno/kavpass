# kavpass 

A simple and quick password generator written in C.

## Install

```console
git clone https://github.com/kavulox/kavpass.git

cd kavpass 

make

doas make install
```

## Usage

```console
kavpass --help
```

## Uninstall



### From source

```console
doas make uninstall
```

### Package Manager

kavpass is available in the [Arch User Repository](https://aur.archlinux.org/packages/kavpass)
