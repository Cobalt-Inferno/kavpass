# kavpass 

A simple and quick password generator written in C.

## Install from Source:
### Arch linux
```console
git clone https://github.com/kavulox/kavpass.git

cd kavpass 

make

doas make install
```

### Debian/Ubuntu
```console
git clone https://github.com/kavulox/kavpass.git

cd kavpass 

make

sudo make install
```
